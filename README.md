# Project Title

Simple Banking Operations 

## Description
Simple Java Server to make basic account management (withdraw + transfer + deposit + inquery) to use RPC.
by using SOAP in Java and generating wsdl and using it by clients.

## Getting Started
* run service publisher first
* run client secondly
